# 🚀 Automatic Analysis System using AI/ML

## Overview

Automatic Analysis System using AI/ML is an intelligent analytics platform that automates data analysis, visualization, and report generation. The system enables users to upload datasets, perform automated preprocessing, generate meaningful insights using Machine Learning techniques, and visualize results through interactive dashboards.

The platform is designed to reduce manual analysis effort and help users make data-driven decisions quickly and efficiently.

---

## Features

- 📊 Automated Data Analysis
- 🤖 AI/ML-Based Insight Generation
- 📈 Interactive Data Visualizations
- 🧹 Data Cleaning and Preprocessing
- 📄 Automated Report Generation
- ⚡ Fast and User-Friendly Interface
- 📉 Trend and Pattern Identification
- 🔍 Exploratory Data Analysis (EDA)

## Installation

### Clone Repository

```bash
git clone https://github.com/SumitTodkar/Automatic-Analysis-System-using-AIML.git
cd Automatic-Analysis-System-using-AIML
```

### Backend Setup

```bash
pip install -r requirements.txt
cd backend
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
